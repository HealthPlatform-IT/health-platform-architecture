## Resumo executivo

**Q-010:** Notification = entrega **in-app** a usuário autenticado; Communication = entrega a **pessoas** por **canais externos**. Critério = superfície de entrega (não a palavra “notificação”). Fan-out permitido. Domain governa preferências; Template Service permanece independente. Integração sistema = fora. **Q-011 fora**.
