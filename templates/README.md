# Templates

Esta pasta contém os modelos oficiais de documentação utilizados no repositório `health-platform-architecture`.

Os templates existem para garantir consistência na criação de documentos arquiteturais, registros de decisão, sessões de discussão, contexto para IA, diagramas, domínios, módulos, entidades e contratos de API.

---

## Objetivo

Padronizar a forma como o conhecimento arquitetural da Health Platform é registrado, revisado e evoluído.

Cada documento importante deve seguir um template adequado ao seu propósito.
